Technology
The following set of technologies was used:
	- ASP.NET Core
	- Entity Framework Core
	- React (для front-end)


A web application for accepting a delivery order with the following functionality:
- New order creation form:
- City of the sender
- Sender's address
- Recipient city
- Address of the recipient
- Cargo weight
- Date of pickup
All fields are required.

- Order list display form: all created orders should be displayed in a separate form. Additional fields, entered by the user when creating an order, should be displayed automatically generated order number
